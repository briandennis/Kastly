## What is it?

A site where you can browse podcasts and combine the episodes you like into playlists. You can also view, comment, and like playlists other users have made. 

## What's it built with?

### Frontend

- React
- Redux
- React Router

### Backend 

- Node JS
- Express
- MongoDB
